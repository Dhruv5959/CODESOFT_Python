Password Generator (Python | Tkinter)

This project is a simple **GUI-based Password Generator** built using Python and Tkinter as part of the CodSoft Python Internship.

Features
- Generate a random password based on user-defined length
- Uses letters, digits, and special characters
- Input validation for incorrect entries
- One-click copy to clipboard
- Simple and user-friendly GUI

Technologies Used
- Python
- Tkinter (GUI)
- random module

How to Run
1. Make sure Python is installed on your system.
2. Clone the repository or download the project files.
3. Run the program using:

```bash
python main.py
