# Rock Paper Scissors Game (Python)

This project is a simple **Rock–Paper–Scissors game** built using Python as part of the **CodSoft Python Internship**.

## Features
- User plays against the computer
- Computer choice is generated randomly
- Validates user input (rock / paper / scissor)
- Determines winner correctly
- Displays score at the end

## Technologies Used
- Python
- random module

## How to Run
1. Make sure Python is installed on your system.
2. Open a terminal in the project directory.
3. Run the program using:

```bash
python main.py
